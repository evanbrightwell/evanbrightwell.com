# EB Website

Local-first personal website project for Evan Brightwell.

This repo is intended to hold the source for a personal portfolio/presence site with a clean interface-led design, clever interaction, and sections for bio, gallery/work, and contact.

No hosting provider or framework has been selected yet.

## Current Direction

- Short first-screen intro.
- Menu into deeper sections: Bio, Gallery, Work, Contact.
- No blog or personal writing section.
- Visual style: clean design interface with thoughtful interactive details.

## Suggested Next Decisions

1. Choose the first build stack: likely Astro, Next.js, or a lightweight Vite app.
2. Decide whether the site should be mostly static or highly interactive.
3. Gather initial gallery/work assets.
4. Pick a deployment target when ready: Vercel, Netlify, or Cloudflare Pages.

## Local Workflow

Use Git locally for versioning. A remote repository can be added later if deploy previews, backup, or automatic deployments become useful.
