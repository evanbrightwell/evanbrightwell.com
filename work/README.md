# Work Library

This folder is the source of truth for portfolio imagery used by the site.
Images outside `work/` are not candidate work images unless they are explicitly copied here.

Use the category/project folders as the editable review surface. If a render is enhanced, replace or add it inside the appropriate `work/<category>/<project>/` folder, then update the site data to point at that file.

- `display-systems/` - Display Systems: 8 projects, 53 images
- `process-systems/` - Process / Systems: 11 projects, 49 images
- `product-packaging/` - Product & Packaging: 14 projects, 46 images
- `spaces/` - Spaces: 9 projects, 54 images
- `archive/` - retired or overflow work grouped by category. Keep it inside `work/` so it remains searchable, but do not wire archived images into the site unless they are promoted back into an active category folder.
